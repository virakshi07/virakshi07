## 💫 About Me

Hi there! 👋 I'm **Virakshi Dhawar**, a Computer Science undergraduate from **Solapur, India** — passionate about building technology that actually means something. I write Python, think in systems, and find my best ideas at the intersection of **Machine Learning** and real-world problem solving.

From training computer vision models to shipping CI/CD pipelines, I love the full journey — ideating, building, breaking, and fixing. I've contributed to open source projects, competed in hackathons, and built AI-powered civic tech that helps first-time voters navigate the electoral process.

I'm driven by curiosity — whether it's exploring a new ML architecture or jumping into a 24-hour hackathon with a half-formed idea. If it's a hard problem and code can help solve it, I'm already interested.

When I'm not coding, I'm probably discovering a new hobby just to see how far I can take it. 🚀

- 🔭 I'm currently working on **VoteSmart-AI**, **DetectX**, and actively shipping across multiple open source projects
- 👥 I'm looking to collaborate on **AI/ML projects** — especially ones tackling real-world problems in civic tech, computer vision, or intelligent systems
- 🤝 I'm looking for help with **MLOps and model deployment** — scaling ML pipelines and serving models in production environments
- 🌱 I'm currently learning **Advanced Deep Learning**, System Design for distributed systems, and Cloud Infrastructure on AWS
- 💬 Ask me about **Python, Machine Learning, Computer Vision, CI/CD automation**, and surviving hackathons
- ⚡ Fun fact: I enjoy turning real-world problems into tech solutions through hackathons and projects
